## pwsh

### prerequisites

1. `oh-my-posh` [here](https://ohmyposh.dev/)
2. `terminal-icons` [here](https://github.com/devblackops/Terminal-Icons)
3. `psreadline` [here](https://github.com/PowerShell/PSReadLine)

### content

| status | command | description        | how to use?                                                                                                                                    |
| ------ | ------- | ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| 游릭     | `mati`  | shutdown your pc   | `$ mati` equals to `$ shutdown -s -t 0`<br>help: `$ get-help mati`                                                                             |
| 游릭     | `ga`    | git: add           | `$ ga` equals to `$ git add .`<br>help: `$ get-help ga`                                                                                        |
| 游릭     | `gc`    | git: commit        | `$ gc` equals to `$ git commit -s`<br>help: `$ get-help gc`                                                                                    |
| 游릭     | `gp`    | git: push          | `$ gp` equals to `$ git push origin {current_branch}`. you can use another branch with an argument `$ gp feature/xxx`<br>help: `$ get-help gp` |
| 游릭     | `touch` | touch an file      | `$ touch {filename}` equals to `$ touch {filename}` <br>help: `$ get-help touch`                                                               |
| 游릭     | `top`   | show your pc stats | `$ top` <br> [here for reference](https://yvez.be/2019/09/01/lets-create-top-for-powershell/)                                                  |
